rtc-dcstream

Node streams2 interface for working with WebRTC data channels. This stream implementation will cater for current data size limits in the WebRTC data channels.

NOTE: Work in Progress

This module is currently an experimental work in progress, so I'd recommend checking out one of the alternative implementation of data channel -> node stream implementations:

  • rtc-data-stream
