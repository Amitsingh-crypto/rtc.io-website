Using rtc-media for WebRTC media capture

In the our previous tutorial we looked at what was involved with capturing media in the browser using the vanilla getUserMedia API.

In this tutorial we will make use of the rtc-media module to do most of the heavy lifting for us. Consider the following code example:

// include the rtc/media module
var media = require('rtc-media');

// now capture media, and once available render to the document body
media().render(document.body);

This example performs exactly the same functionality (with a few additional smarts) as the code in our previous example.
